# 돌봄 사각지대에 놓인 사회적약자를 위한 상황인지형 감정케어 로봇 Pengkin 🤖

## 1. 개요
### 1.1 주제 선정 배경 🚣
- 사회적 관계 부족
- 헬스케어 관심 및 중요성 증가
- 돌봄 사각지대에 놓인 사람들 (아동, 노인, 장애인)

###  1.2 프로젝트 설명 
- 펭귄 + Kind = Pengkin
- 상황인지형 감정케어 로봇 펭킨은 아두이노 센서 및 라즈베리파이를
활용하여 돌봄사각지대에 놓여진 소외계층들을 정서적 및 신체적으로
보살핌으로써 삶의 질을 제고하고 주 돌봄자에게 효율적인 보탬을
도모하고자 기획함
